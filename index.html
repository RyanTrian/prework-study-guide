HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Declaration is not case-sensitive, while elements is case case-sensitive
  • id is used as a way to identify element

CSS

  • * is a selector that apply the properties to ALL elements on the page
  • CSS style: inline CSS, internal CSS style sheet, and external CSS style sheet
  • display: block; image starts on a new line
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add all changes to your next commit
  • git commit -m "[descriptive message]": commit your staged content as a new commit snapshot
  • git merge [branch]: merge the specified branch's history into the current one
  • git pull origin main: sync local branch with the remote branch in GitHub
